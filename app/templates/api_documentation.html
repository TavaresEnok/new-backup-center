🔒 API do Backup Center

Documentação Interativa e Simples

🔑 Como Começar

Você precisa de um Token de API para usar esta API. Não tem um ainda?

Crie um token em: Configurações → API Tokens

Depois, use o token no header:

Authorization: Bearer bc_seu_token_aqui

📋 3 Passos para Integrar

1

Encontre seus Grupos (Pastas/Clientes)

Esta chamada retorna a lista de todos os grupos/pastas que seu token pode acessar.

GET /api/v1/external/groups
ℹ️ O que procurar na resposta: Cada grupo terá um campo "id" (um código UUID). O device_count considera apenas dispositivos cujo último backup está em sucesso.

Exemplo de resposta:

{ "total": 2, "groups": [ { "id": "91d21c58-dc9c-4e39-b008-b6ff7aa7f53a", "name": "Flashnet", "device_count": 10, "last_backup_at": "2026-05-14T10:30:00" } ] }
2

Veja os Backups de um Grupo

Agora use o ID do grupo do Passo 1 para listar os backups disponíveis dos dispositivos cujo último backup está em sucesso.

GET /api/v1/external/groups/{GROUP_ID}/backups?status=success

Parâmetros:

  • GROUP_ID → Copie o ID do Passo 1
  • status → Use "success" para ver apenas backups bem-sucedidos; dispositivos cujo último backup falhou não são expostos
  • page → Para paginação (padrão: 1)
  • per_page → Quantos por página (padrão: 50, máximo: 200)
ℹ️ O que procurar na resposta: Procure o campo "id" (do backup) e "device_id". Estes são os identificadores únicos que você usará no Passo 3.

Exemplo de resposta:

{ "group": "Flashnet", "page": 1, "total": 5, "items": [ { "id": "123e4567-e89b-12d3-a456-426614174000", "device_id": "bfa9d28a-4fc6-4dc4-9280-a2aad70feeb3", "device_name": "FLASHNET - ROUTER - HW NE8000 BORDA", "status": "success", "file_size_bytes": 1024000, "hash_sha256": "abc123def456...", "created_at": "2026-05-14T09:15:00", "download_url": "/api/v1/external/backups/123e4567-e89b-12d3-a456-426614174000/download" } ] }
3

Faça o Download do Arquivo

Use o ID do backup do Passo 2 para baixar o arquivo binário.

GET /api/v1/external/backups/{BACKUP_ID}/download

Parâmetros:

  • BACKUP_ID → Copie o ID do backup do Passo 2
✅ Regra Importante: Apenas backups com status: "success" podem ser baixados. Se o backup está em outro status, nenhuma ação ocorrerá.

💻 Exemplos de Código

Python (usando requests)

import requests TOKEN = "bc_seu_token_aqui" BASE_URL = "https://backupcenter.ajustconsulting.com.br" headers = {"Authorization": f"Bearer {TOKEN}"} # Passo 1: Listar grupos response = requests.get(f"{BASE_URL}/api/v1/external/groups", headers=headers) groups = response.json()["groups"] group_id = groups[0]["id"] # Passo 2: Listar backups do grupo response = requests.get( f"{BASE_URL}/api/v1/external/groups/{group_id}/backups", params={"status": "success"}, headers=headers ) backups = response.json()["items"] backup_id = backups[0]["id"] # Passo 3: Download response = requests.get( f"{BASE_URL}/api/v1/external/backups/{backup_id}/download", headers=headers ) with open("backup.bin", "wb") as f: f.write(response.content)

cURL (linha de comando)

# Passo 1 curl -H "Authorization: Bearer bc_seu_token_aqui" \ https://backupcenter.ajustconsulting.com.br/api/v1/external/groups # Passo 2 (substitua GROUP_ID) curl -H "Authorization: Bearer bc_seu_token_aqui" \ "https://backupcenter.ajustconsulting.com.br/api/v1/external/groups/GROUP_ID/backups?status=success" # Passo 3 (substitua BACKUP_ID) curl -H "Authorization: Bearer bc_seu_token_aqui" \ https://backupcenter.ajustconsulting.com.br/api/v1/external/backups/BACKUP_ID/download \ -o backup.bin

JavaScript (Fetch API)

const TOKEN = "bc_seu_token_aqui"; const BASE_URL = "https://backupcenter.ajustconsulting.com.br"; // Função auxiliar async function apiCall(endpoint, options = {}) { const response = await fetch(BASE_URL + endpoint, { ...options, headers: { "Authorization": `Bearer ${TOKEN}`, ...options.headers } }); return response.json(); } // Passo 1 const groupsData = await apiCall("/api/v1/external/groups"); const groupId = groupsData.groups[0].id; // Passo 2 const backupsData = await apiCall( `/api/v1/external/groups/${groupId}/backups?status=success` ); const backupId = backupsData.items[0].id; // Passo 3 const response = await fetch( `${BASE_URL}/api/v1/external/backups/${backupId}/download`, { headers: { "Authorization": `Bearer ${TOKEN}` } } ); const blob = await response.blob(); // Salvar arquivo...
